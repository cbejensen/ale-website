This directory contains the library of public resources for the main website.

Page content can be found within the view/ directory.
Styles related to the pages themselves can be found within the css/ directory.
Client-side code related specifically to the pages can be found in the js/ 
directory.

Other resources, such as default styles, site-wide styles, site-wide js, layout 
templates, etc. are located in the project's resources/ folder.
